These three libraries need to be installed in order for the sketch to compile and work correctly.  To install...download each and unzip each one at a time within the "libraries" directory of your Arduino IDE installation.

To download these libraries from the vendor directly, please use the following urls:
Adafruit_NeoPixel Source: https://github.com/adafruit/Adafruit_NeoPixel
Adafruit_PWMServoDriver Source: https://github.com/adafruit/Adafruit-PWM-Servo-Driver-Library
BHD's SlowServoPCA9685 Source: https://drive.google.com/file/d/0B5B8A65frsBgZ3VpeGxpM1lzaFE/edit  Thanks BHD!
